Parking App Help

How to Use the Parking Management App

Features Overview

  • Add Cars: Assign a car to an available parking space.
  • Update Status: Modify the ticket status of a parked vehicle.
  • Remove Cars: Remove a car using the ‘Reindeer’ role. Mistake? Use ‘Undo’ (local-only).
  • Manage Slots: Change parking slot capacity (admin access required).

Smart Display

Only occupied parking spots are shown, keeping the interface clear.
